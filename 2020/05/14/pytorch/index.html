人工智能部分相关框架的安装

五月 14, 2020

前言

前几天因为人工智能导论作业要求安装 pytorch 框架来做一些实验，还有报名参加了一个学校的人工智能算法挑战赛，所以在主机上安装了 pytorch 跟 tensorflow。摸索出来的安装过程还是比较漫长的。

安装方式

0. 安装平台及方案
  • 系统：win 10
  • 方案：在主机上通过 python+anaconda 来创建 pytorch/tensorflow 的虚拟环境
  • 提示：如果仅仅只需要安装 tensorflow，建议直接跳转 Tensorflow 官网 install 页面
1. 安装 python（这部分就不多赘述了）
2. 安装 anaconda
  • 下载 anaconda 镜像，这里因为去官网下载会有速度上的限制，所以推荐去清华镜像站下载：
  • 清华镜像站
  • anaconda 下载页面
  • 这里建议先按更新日期排序，找到最新的 anaconda3（别下到 anaconda1 了）然后选择 64 位版本的.exe 文件
  • 剩余的配置方面我就不多说了，毕竟装 python 的有很多都是 python+anaconda
3. 安装 pytorch

  • 用管理员权限打开 anaconda promt（默认位置都是 c:\…….的位置，根据个人需要调整设置）

  • 创建好文件夹（根据需要执行这一步）

  • 输入命令 conda create --name pytorch python=3.6 anaconda，来建立 pytorch 的虚拟环境(命令中的 pytorch 是虚拟环境的名称)

  • 创建完成后，输入 activate pytorch 进入虚拟环境，可以通过输入命令 conda env list ，看到当前系统 anaconda 下所有的虚拟环境，其中 base 是 anaconda 本身带有的，pytorch 是新建的，后边括号中有星星（*）的就是当前所在环境

  • 此时还可以通过输入 python 来确认是否成功进入环境中，若出现警告 ⚠，则需要根据警告内容检查配置过程，成功的应该是跟在 idle 中的界面内容一样的

  • 然后就是安装 pytorch 相关的库了，此处可以在官网安装也可以从别的源安装，这里同样的推荐从清华源安装，官网的速度简直不能恭维，但是可以从pytorch 官网 处确认你需要的配置

  • 切换源这里转载一个网站给大家，我就不重复别人的话了，《 使用国内镜像源安装 pytorch 》

  • 设置好镜像源就可以进行下一步了，输入命令 conda install pytorch torchvision cudatoolkit=10.2 ，其中 cudatookit 这个东西的需求及根据个人情况调整，前两个是必要的，然后能做的就是等待了，顺便吐槽一句，这里麻烦的地方就是这个源也没有想像中的稳定，如果出现下载失败的情况，一定要去 anaconda 放包的地方把 pytorch 有关的文件夹和压缩包都删除掉，不然下载成功的那次可能会出现失败的情况，不知道怎么找的可以再运行一次上面的安装命令，会显示一个 warning 提示你去哪里手动删除

  • 最后就是安装成功后的测试了，输入 python 命令，

    有版本输出信息即为安装成功

  • 最后通过 conda deactivate 关闭虚拟环境

4. 安装 tensorflow2.0
  • tensorflow 相对于 pytorch 的安装真实地便利了许多，因为有一个类似于安装向导的网站相当于手把手教你安装，贴在下面
  • Tensorflow 官网 install 页面（其实详细的官网说得很清楚了，无可挑剔的那种，本人一次成功，所以我也不多说什么）
